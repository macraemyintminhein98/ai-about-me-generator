# AI Profile Crafter

Generate professional 'About Me' sections for LinkedIn, resumes, and portfolios using AI. Simply input your skills, industry, role, and achievements, and get instant, tailored profile summaries.

## Features:
- AI-powered content generation
- Multiple variations for different platforms
- Easy copy-to-clipboard functionality

## How to Use:
1. Enter your keywords (e.g., "Software Engineer, Python, AWS, Led team, 10% efficiency gain").
2. Click 'Generate Profiles'.
3. Copy your desired 'About Me' section.

Ideal for job seekers, freelancers, and professionals looking to quickly enhance their online presence.
